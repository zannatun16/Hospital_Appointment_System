# Hospital_Appointment_System

> A modern web-based Hospital Appointment Management System built with **PHP** and **MySQL** using the **MVC Architecture**.

## 📖 Overview

The Hospital Appointment Management System is designed to simplify the appointment process between patients and doctors. It provides separate dashboards for administrators, doctors, and patients, making hospital management more organized, efficient, and user-friendly.



## ✨ Features

### 👨‍💼 Admin

* Manage Doctors
* Manage Patients
* Manage Departments & Specializations
* View & Manage Appointments
* Dashboard & Reports

### 👨‍⚕️ Doctor

* Secure Login
* View Scheduled Appointments
* Update Profile
* Manage Availability

### 👤 Patient

* Register & Login
* Search Doctors
* Book Appointments
* View Appointment History
* Update Profile
